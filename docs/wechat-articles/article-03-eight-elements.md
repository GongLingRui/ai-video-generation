## AI 视频为什么总让人失望？

用过 AI 视频工具的人都知道，最痛苦的不是"生成得慢"，而是"生成得不对"。

你写了一句 prompt：

> "一个女生在咖啡馆里看书。"

AI 给你生成了一个女生在咖啡馆里看书的视频。看起来没问题？但仔细看 —— 她坐在哪个位置？光线从哪来？背景里有什么？镜头是固定还是运动？穿什么衣服？看什么书？

全部随机。

这就是 AI 视频的"抽卡"本质：**你的 prompt 越模糊，AI 的自由度越大，结果偏离你想象的可能性就越高**。

传统的解决办法是"加更多描述词"。但描述词一多，AI 的注意力就被稀释了，反而更容易忽略重点。

八要素系统的思路完全不同：**不是让 prompt 更长，而是让描述更有结构**。

## 八要素详解

### 1. 主体（Subject）

主体是画面的核心。不是简单写"一个女生"，而是要精确到：

- 外貌特征：年龄、发型、服装、配饰
- 情绪状态：微笑、沉思、焦虑、疲惫
- 在画面中的重要性：占据主导还是融入环境

示例："年轻亚裔女性，齐肩黑发，穿米色针织开衫，低头专注阅读，神情宁静。"

### 2. 构图（Composition）

构图决定了观众看哪里。包括：

- 景别：远景、全景、中景、近景、特写
- 角度：平视、俯拍、仰拍、鸟瞰
- 位置：主体在画面中的相对位置
- 景深：背景清晰还是虚化

示例："中景，平视角度，主体位于画面右侧三分之一处，浅景深虚化窗外街景。"

### 3. 光线（Lighting）

光线是氛围的灵魂。包括：

- 光源方向：顺光、侧光、逆光、顶光
- 光质：硬光（明暗对比强烈）还是柔光（过渡自然）
- 色温：暖光（橙黄）还是冷光（蓝白）
- 时间感：清晨、正午、黄昏、夜晚

示例："侧逆光从右侧大窗户射入，形成温暖的轮廓光，室内主光源为柔和的暖色顶灯。"

### 4. 背景（Background）

背景不是"后面的东西"，它是叙事的延伸。包括：

- 环境类型：室内/室外，具体空间特征
- 时代感：现代、复古、未来、奇幻
- 空间层次：前景遮挡、中景主体、远景延伸
- 重要道具：哪些物品需要出现在画面中

示例："现代独立咖啡馆，深色木质家具，背景书架上有少量绿植，窗外可见模糊的街景。"

### 5. 动作/运动（Action/Movement）

这是最容易被忽略、却最能提升画面质感的要素。包括两个层面：

- 主体动作：人物在做什么、怎么做、速度如何
- 镜头运动：固定、推、拉、摇、移、跟、升、降

示例："主体右手轻翻书页，头微微侧倾；镜头以极缓慢的速度向前推近，从腰景推至胸景。"

### 6. 文字叠加（Text Overlay）

有时候画面需要文字信息：

- 字幕/对白
- 时间地点标注
- UI 界面元素（如果是科幻/现代题材）
- 品牌标识

示例："画面右下角半透明字幕：'1998年，台北'。"

### 7. 转场细节（Transition）

前后镜头怎么衔接，直接影响叙事的流畅度：

- 硬切：最常用的直接切换
- 叠化：两个画面短暂重叠，暗示时间流逝或回忆
- 淡入淡出：常用于段落开头和结尾
- 匹配剪辑：利用相似形状/动作连接两个不同时空的镜头

示例："硬切至下一个镜头，保持室内暖色调的连续性。"

### 8. 音频（Audio）

虽然视频生成工具目前大多不直接生成音频，但在分镜阶段规划好音频，能为后期配音/配乐提供明确指导：

- 环境音：咖啡馆的咖啡机声、轻轻的爵士乐
- 音效：翻书声、窗外的雨声
- 音乐情绪：慵懒、紧张、悲伤、 hopeful

示例："背景播放慵懒的爵士钢琴，偶尔混入咖啡杯轻触碟子的清脆声响。"

## 八要素为什么对 AI 特别有效？

传统的 AI 视频 prompt 是一长串自然语言描述。AI 模型（尤其是基于文本生成视频的大模型）在处理长文本时，对不同部分的注意力权重是不一样的。你写在后面的细节，往往会被前面的主语"淹没"。

八要素系统的优势在于：

**1. 结构化输入**：AI 模型（特别是经过工具调用 fine-tune 的模型）对结构化数据的理解远好于自由文本。每个要素都有明确的语义边界。

**2. 可验证性**：你可以快速检查"这个动作要素是不是漏了？""构图和光线有没有冲突？"，而不需要从一大段文字里找。

**3. 可复用性**：当你建立了一个角色的"主体"描述后，可以直接复制到所有包含这个角色的镜头里，保证一致性。

**4. 可学习性**：新手通过观察 AI 生成的八要素，能快速理解专业分镜的思考方式。

## 实战案例

我们来对比一个 prompt 的两种写法：

**写法 A（传统）：**

> "一个年轻女孩在夕阳下的海边奔跑，风吹起她的长发，镜头跟着她移动，画面很唯美。"

**写法 B（八要素）：**

| 要素 | 描述 |
|------|------|
| 主体 | 20 岁左右的东亚女性，及腰长发，穿白色连衣裙，赤脚奔跑，面带笑容 |
| 构图 | 全景到中景，侧面跟拍 |
| 光线 | 黄金时刻侧逆光，海面反射金色波光 |
| 背景 | 开阔海滩，远处有模糊的礁石和灯塔 |
| 动作 | 主体以中等速度沿 shoreline 奔跑，头发和裙摆随风扬起；镜头用斯坦尼康平稳跟拍，略带轻微呼吸感 |
| 文字 | 无 |
| 转场 | 叠化至下一个镜头（暗示时间流逝） |
| 音频 | 海浪声、风声、轻快的原声吉他 |

哪个生成结果更接近创作者的想象？不言而喻。
