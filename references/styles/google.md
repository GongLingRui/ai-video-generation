# Google发布会风格模板

## 色彩方案
- **primary**: #4285F4 (Google蓝)
- **secondary**: #34A853 (Google绿)
- **background**: #FFFFFF (纯白背景)
- **accent**: #EA4335 (Google红)

## 构图规则
- 极简主义，大量留白
- 主体居中或三分法构图
- 避免复杂前景和背景
- 使用高对比度确保文字可读性
- 干净的边框和分隔线

## 视觉特征
- 商务蓝为主色调
- 清晰的图标和插图
- 平滑的转场动画
- 干净的字体排版
- 高对比度、高清晰度

## 适用场景
- 产品发布会
- 企业宣传片
- 科技产品介绍
- 商务演示视频
- 教育培训内容

## 提示词模板

### 视频生成
```
{主体描述}。Google发布会风格，极简商务蓝配色，大量留白，清晰图标，高清质感。
```

### 首帧生成
```
{静态描述}。极简构图，商务配色，纯白背景，4K超高清。
```

### 尾帧生成
```
{结果状态}。保持极简风格，为下一镜头留出空间。
```

## 示例

**主体**: 一位穿蓝色连衣裙的年轻舞者在白色背景舞台上独舞

**视频Prompt**:
```
一位穿蓝色连衣裙的年轻舞者在白色背景舞台上独舞，特写镜头聚焦面部表情，柔和的顶光照射，Google发布会风格，极简商务蓝配色，大量留白，清晰图标，高清质感。
```

**首帧Prompt**:
```
静态特写镜头，年轻舞者蓝色连衣裙，白色背景舞台，柔和顶光，商务蓝配色，极简构图，4K超高清。
```

**尾帧Prompt**:
```
舞者完成旋转动作后的表情，蓝色连衣裙摆动的瞬间，舞台灯光渐强，保持极简风格，为下一镜头做铺垫。
```
