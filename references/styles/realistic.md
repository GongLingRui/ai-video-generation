# 写实风格模板

## 色彩方案
- **primary**: #4A4A4A (中性灰)
- **secondary**: #808080 (浅灰)
- **background**: #F5F5F5 (浅灰背景)
- **accent**: #FF6B35 (橙色点缀)

## 构图规则
- 真实感构图
- 自然光照效果
- 写实色彩还原
- 真实场景细节
- 自然透视关系

## 视觉特征
- 写实风格
- 真实感
- 自然光线
- 真实场景
- 高保真画质

## 适用场景
- 纪录片
- 真实场景视频
- 产品实拍
- 真实风格宣传
- 教育演示视频

## 提示词模板

### 视频生成
```
{主体描述}。写实风格，真实场景，自然光照，写实色彩还原，真实感，高清画质。
```

### 首帧生成
```
{静态描述}。写实风格，真实场景，自然光线，4K超高清。
```

### 尾帧生成
```
{结果状态}。保持写实风格，为下一镜头营造真实过渡感。
```

## 示例

**主体**: 一位咖啡师在咖啡厅制作拿铁

**视频Prompt**:
```
一位专业咖啡师在现代咖啡厅制作拿铁咖啡，写实风格，真实场景，自然光照，写实色彩还原，真实感，高清画质。
```

**首帧Prompt**:
```
静态特写，咖啡师专注表情，咖啡厅环境，写实风格，自然光线，4K超高清。
```

**尾帧Prompt**:
```
拉花完成后的完美拿铁，保持写实风格，为下一镜头营造真实过渡感。
```
