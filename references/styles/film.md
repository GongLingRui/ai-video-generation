# 电影风格模板

## 色彩方案
- **primary**: #8B7355 (电影棕)
- **secondary**: #2C3E50 (深蓝灰)
- **background**: #1C1C1C (深黑背景)
- **accent**: #D4AF37 (金色)

## 构图规则
- 电影感构图，遵循三分法
- 深度景深，背景虚化
- 电影级调色，高对比度
- 专业的光影效果
- 电影镜头语言

## 视觉特征
- 电影质感
- 电影调色
- 专业布光
- 电影镜头语言
- 高动态范围

## 适用场景
- 电影预告片
- 剧情片片段
- 纪录片风格
- 电影感宣传片
- 艺术短片

## 提示词模板

### 视频生成
```
{主体描述}。电影风格，专业电影布光，电影级调色，景深效果，高清质感，电影感。
```

### 首帧生成
```
{静态描述}。电影质感，专业构图，电影布光，4K超高清。
```

### 尾帧生成
```
{结果状态}。保持电影风格，为下一镜头营造电影过渡感。
```

## 示例

**主体**: 一位侦探在雨夜的街道上行走

**视频Prompt**:
```
一位穿着风衣的侦探在雨夜的街道上行走，电影风格，专业电影布光，电影级调色，景深效果，雨滴反光，电影感。
```

**首帧Prompt**:
```
静态远景，侦探雨夜街道，电影质感，专业构图，电影布光，4K超高清。
```

**尾帧Prompt**:
```
侦探停下脚步转身回望的瞬间，保持电影风格，为下一镜头营造电影过渡感。
```
