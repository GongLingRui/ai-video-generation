# 动漫风格模板

## 色彩方案
- **primary**: #FF6B9D (粉色)
- **secondary**: #4ECDC4 (青色)
- **background**: #FFF5F8 (浅粉背景)
- **accent**: #FFE66D (黄色)

## 构图规则
- 二次元角色居中构图
- 明亮的高饱和度色彩
- 清晰的线条和轮廓
- 夸张的表情和动作
- 梦幻的背景效果

## 视觉特征
- 日本动漫风格
- 明亮的色彩搭配
- 可爱的角色设计
- 梦幻的场景背景
- 二次元美学

## 适用场景
- 动漫风格视频
- 卡通角色介绍
- 游戏宣传视频
- 少女向内容
- 梦幻主题视频

## 提示词模板

### 视频生成
```
{主体描述}。日本动漫风格，高饱和度色彩，清晰的线条，明亮的画面，可爱梦幻的氛围。
```

### 首帧生成
```
{静态描述}。动漫风格，明亮色彩，居中构图，高清画质，4K超高清。
```

### 尾帧生成
```
{结果状态}。保持动漫风格，为下一镜头留下梦幻过渡空间。
```

## 示例

**主体**: 一位粉色头发的少女在樱花树下

**视频Prompt**:
```
一位粉色头发的少女在樱花树下微笑，日本动漫风格，高饱和度色彩，清晰的线条，明亮的画面，樱花飘落的梦幻氛围。
```

**首帧Prompt**:
```
静态特写，粉色头发少女，樱花树下，动漫风格，明亮色彩，居中构图，4K超高清。
```

**尾帧Prompt**:
```
少女伸手接住樱花花瓣的瞬间，保持动漫风格，樱花飘落的梦幻过渡效果。
```
